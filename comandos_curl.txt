Login
curl -X POST http://localhost:8000/api/auth/login/ -H "Content-Type: application/json" -d "{\"username\": \"YOUR_USERNAME\", \"password\": \"YOUR_PSWD\"}"

Logout
curl -X POST http://localhost:8000/api/auth/logout/

Try the token
curl -H "Authorization: Token YOUR_TOKEN" http://localhost:8000/api/auth/test/

Create an user
curl -X POST http://localhost:8000/api/auth/signup/ -H "Content-Type: application/json" -d "{\"username\": \"YOUR_USERNAME\", \"password\": \"YOUR_PSWD\"}"
